codeigniter-base-model

Codeigniter Base Model to klasa, która rozszerza podstawową klasę modelu w Codeigniterze, czyli CI_Model. Zawiera pełny zakres funkcji CRUD do szybkich interakcji z bazami danych a także system obserwatorów, walidację danych w modelu, inteligentne odgadywanie nazw tabli oraz miękkie kasowanie rekordów.

Instalacja

1. Ściągnij najnowszą wersję biblioteki z https://raw.githubusercontent.com/jamierumbelow/codeigniter-base-model/master/core/MY_Model.php

2. Umieść plik w katalogu application/core . Klasa będzie uruchamiać się automatycznie.

Zastosowanie

Aby nasze modele od razu otrzymały funkcjonalności zawarte w Codeigniter Base Model wystarczy rozszerzyć nasze modele o MY_Model, zamiast jak do tej pory o CI_Model. 

<?php
class Page_model extends MY_Model
{
}

// Lokalizacja: application/models/page_model.php

Następnie w kontrolerze ładujemy plik modelu i odwołujemy się do odpowiedniej metody lub łączymy parę metod w całość. 

<?php if ( ! defined('BASEPATH')) exit('No direct script access allowed');

class Welcome extends CI_Controller {

    function __construct()
    {
        parent::__construct();
        $this->load->model('page_model');
    }

    public function index()
    {
        $data['pages']=$this->page_model->order_by('created_at','desc')->get_all();
        $this->load->view('welcome_message',$data);
    }
}

/* Lokalizacja: ./application/controllers/welcome.php */

Z kolei w widoku wyświetlamy pobrane dane. 

<?php 
foreach($pages as $value){
    echo $value->title;
    echo $value->description;
}
/* Lokalizacja: ./application/views/welcome-message.php */
